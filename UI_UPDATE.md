# 网站优化说明

## ✨ 优化概述

本次优化将个人网站升级为**"智能体研究空间"**，体现**温度理念**和**星辰闪耀**的主题。

---

## 🎨 核心优化

### 1. 星空背景优化（温度理念）

#### 温暖色系
- **星星颜色**：金色、橙色、淡粉色、玉玉米色等温暖色系
- **背景渐变**：深棕 → 深紫 → 深蓝 → 黑，营造温暖深邃感
- **光晕效果**：大星星带有温暖的光晕，增强温度感

#### 闪烁效果增强
- 闪烁幅度：0.3 → 0.7（更明显）
- 闪烁速度：0.01 → 0.04（更活跃）
- 透明度：30% → 100%（更明亮）

### 2. 星辰闪耀效果

#### 流星效果
- ✨ 随机生成流星（2% 概率）
- 流星尾巴渐变消失
- 流星头部闪烁
- 金色流星，符合温度理念

#### 星云效果
- 🌌 添加 2 个星云区域
- 暖色调：橙色、金色调
- 柔和的渐变光晕
- 营造梦幻氛围

#### 银河效果
- 旋转速度：0.0008（更快）
- 条数：5 → 8（更璀璨）
- 颜色：暖色渐变（金色到橙色）
- 透明度增强（更明显）

## 🎯 设计理念

### 温度理念
1. **温暖色系**：金色、橙色等暖色
2. **柔和光晕**：星星带有温暖的光晕
3. **舒适体验**：不刺眼，温和的视觉感受

### 星辰闪耀
1. **动态效果**：流星、闪烁、旋转
2. **璀璨星光**：增强的银河、星云
3. **无限光芒**：每个智能体都是独特的星辰

### 智能体研究空间
1. **专业性**：研究、探索、创新
2. **未来感**：AI、星辰、未来
3. **温度感**：每颗星辰都有温度

## 📝 使用说明

### 体验效果
访问 `http://localhost:5000` 或部署后的 `http://tianqizhin.top`

### 修改理念文案
如果需要修改理念文字，编辑 `src/app/page.tsx`：
```tsx
// 理念卡片
<p className="text-lg text-amber-50/70 font-light leading-relaxed italic">
  "由我打造的智能体，将会像星辰一般闪耀下去"
</p>
```

### 调整颜色
如果需要调整颜色，编辑 `src/app/page.tsx` 中的颜色数组：
```tsx
const warmColors = [
  "#FFD700", // 金色
  "#FFA500", // 橙色
  "#FFB6C1", // 淡粉色
  // 添加更多颜色...
];
```
